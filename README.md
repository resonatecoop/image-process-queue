# Image processing

Convert image files to jpeg, webp following a set of variants

## Install

```sh
npm install
```

## Build (babel)

```sh
npm run build
```

## Env

```sh
cp .env.example .env
```

## Usage

```sh
node ./lib/index.js run --name my-queue-name
```

## Test

```sh
npm test
```
